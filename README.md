Sass Starter
============

### To Use

* You will need Node, Grunt, and Bundler
* From the command line, cd into the folder
* Run "npm install"
* Run "bundle install"
* Run "grunt serve"
* To stop watching, use Ctrl+C

To create a custom modernizr build based on your code, run "grunt modernizr:dist".
